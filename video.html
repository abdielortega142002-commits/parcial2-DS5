Video sobre la Web Application Firewall (WAF)


En este video explica que un WAF (Web Application Firewall) es una herramienta de seguridad que se coloca entre una aplicación web y el tráfico de internet para filtrar, monitorear y bloquear solicitudes maliciosas, protegiendo así la aplicación contra ataques comunes como la inyección SQL o el cross-site scripting (XSS). A diferencia de los firewalls tradicionales que protegen la red, el WAF protege la capa de aplicación (nivel 7 del modelo OSI), donde ocurren la mayoría de los ataques a sitios web. Además, se menciona que puede implementarse como hardware, software o servicio en la nube, y que su efectividad depende de una configuración adecuada y reglas actualizadas para adaptarse a las amenazas cambiantes.
