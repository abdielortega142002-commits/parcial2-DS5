¿Qué es Web Application Firewall (WAF)?


Una Web Application Firewall (WAF) es una tecnología de seguridad que filtra y supervisa el tráfico HTTP o HTTPS entre una aplicación web y el resto de Internet para bloquear ataques dirigidos a la capa de aplicación (capa 7 del modelo OSI), como inyección de consultas SQL, Cross‑Site Scripting (XSS), inclusión de archivos y cross-site request forgery.


Al desplegar un WAF en una aplicación web, se coloca un escudo entre la aplicación web e Internet. Si bien un servidor proxy protege la identidad del equipo del cliente por medio de un intermediario, un WAF es un tipo de proxy inverso que protege al servidor de los riesgos al hacer que los clientes atraviesen el WAF antes de llegar al servidor.


El WAF opera por medio de un conjunto de reglas, normalmente denominadas directivas. Estas directivas tienen el fin de proteger contra vulnerabilidades en la aplicación al filtrar el tráfico malicioso. El valor de un WAF procede, en parte, de la velocidad y facilidad con que se pueden aplicar modificaciones en las directivas que permiten una respuesta más rápida ante diversos vectores de ataque; durante un ataque DDoS, se puede implementar rápidamente la limitación de velocidad modificando las directivas del WAF.

¿Cómo es el Funcionamiento del WAF?

  • Analiza las peticiones HTTP/HTTPS entrantes y salientes buscando patrones maliciosos.
  • Analiza firmas conocidas y anomalías en el comportamiento del tráfico.
  • Aplica reglas predefinidas y reglas personalizadas basadas en la lógica de la aplicación para permitir bloquear o desafiar ciertas solicitudes.
  • Usan modelos de comportamiento o aprendizaje automático para detectar tráfico anómalo y reducir falsos positivos.

¿Cúal es el Propósito Principal del WAF?

De entre los propositos principales de el Web Application Firewall (WAF) podemos mencionar:

  • Proteger las Aplicaciones Web de Ataque Malintensionados
  • Prevenir Explotación de Vulnerabilidades
  • Mantener disponibilidad y Continuidad
  • Reducir Riegos Operativos
